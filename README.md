GalliMart – Hyperlocal Android Application

## 🚀 Overview
GalliMart is a hyperlocal Android application designed to connect nearby users with local markets. It enables users to browse products and place orders seamlessly.

## 📱 Features
- 🛍 Product discovery from local vendors
- 📦 Ordering system
- 🔄 Real-time data handling using Firebase
- 🎨 Clean and user-friendly UI

## 🛠 Tech Stack
- Java
- Android (XML UI)
- Firebase Realtime Database
- Supabase

## 🎯 Objective
To build a practical solution that bridges the gap between local sellers and nearby customers using a mobile platform.

## 📌 Future Improvements
- Payment gateway integration
- Order tracking system
- Improved UI/UX enhancements
